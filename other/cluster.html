清北科技组网计划 #

1/概述 #

清北科技目前的组网方案：

使用 WireGuard 组建公网集群，搭配宝塔 Linux 面板实现 LNMP 负载均衡、MySQL主从复制、Redis主主复制等。

搭建公网集群的意义：

使用云厂商的内网互联服务，的确可以简单快捷地搭建内网集群，然而清北科技和 IURT 以及其他合作组织的服务器分布在世界各地，无法简单通过该方法互联。

使用 WireGuard 等技术，使我们能够把分布在各地的服务器连接为一个有机的整体，让闲置的服务器活起来。

通过公网集群，网站可以共用一个入口IP，再由服务器分发至各个后端节点；入口服务器设在境外，这样即使后端计算节点在境内也能轻松利用其资源；即使是 NAT 服务器，也可以使用相同方法搭建网站。

2/要求 #

加入的每一台服务器都将折价计算为赞助，每位参与者的名字都将出现在赞助榜上。

加入的服务器分为完整节点和基本节点两种。

完整节点：

  • 至少一个公网IP（不包括DDNS）
  • 至少1核1G
  • 拥有完整Root权限（不包括Docker等容器环境）
  • 全端口可访问

基本节点：

  • NAT公网（包括DDNS）
  • 至少1核512MB
  • 拥有受限用户权限（例如Docker版宝塔Linux面板）
  • 可使用厂商提供的端口转发

你可以选择内网IP网段：

  • 192.168.100.1/24：完整节点
  • 192.168.101.1/24：基本节点
  • 192.168.10x.1/24：自定义（如一次性加入3台及以上，可自定义内网网段）
  • 10.0.x.1/24：特殊节点（如仅用于云数据库，仅用于容灾备份等）

加入组网计划后，我们需要执行：

  • 登录SSH，安装并配置WireGuard
  • 安装宝塔面板Linux面板，升级为企业版
  • 安装哪吒面板Agent，实时监控节点状态
    • 清北科技节点监控：访问

完成组网操作，后续操作由服务器地域不同而有差异。

加入组网计划的服务器将免费享受清北科技、IURT及其合作组织的服务器运维和安全管理服务。

目前，已有10台服务器参与组网计划。

加入组网≠捐赠服务器，更像是共用；我们将使用加入的服务器作为计算节点，加入者同样可以使用部分节点。 加入后可免费享受清北科技开发者待遇等多项服务，快来联系我们吧！
